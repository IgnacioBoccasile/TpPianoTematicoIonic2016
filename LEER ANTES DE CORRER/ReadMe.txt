EN CASO DE HABER DESCARGADO EL ARCHIVO DESDE GITHUB, SIGA LOS SIGUIENTES PASOS:

1) Eliminar Archivos sin uso en "js" ( services.js ) y en "templates" ( tab-account.html, tab-dash.html, tab-chats.html ).

2) Añadir plataforma android: ionic platform add android.

3) Instalar en consola NativeAudio: cordova plugin add https://github.com/SidneyS/cordova-plugin-nativeaudio.git

4) Instalar Driver Motorola ( para los dispositivos Motorola ).
